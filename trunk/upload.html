Note to users

Upload a CSV file containing the Order ID and the Order reference, like this:

orderid,orderreference
26718659,26015985
26709170,26006505
26753279,26050568
26709176,26006511
...

The first column must be the Order ID. The second column must be the Order Reference. They must not be quoted. They must be separated by commas. Nothing else matters.

Note to admins

The order file is saved on the server under the chargeback/chargeback/order/order-*. They ought to be simple pipe-delimited text files. You may delete old files if required.
